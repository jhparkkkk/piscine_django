Ex00 : Markdown Cheatsheet

Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

Emphasis

*italic* or _italic_
**bold** or __bold__
~~strikethrough~~

Lists

Unordered

- Item 1
- Item 2
  - Subitem 1
  - Subitem 2

Ordered

1. Item 1
2. Item 2
3. Item 3

Links

[Link text](http://example.com)

Images

![Alt text](http://example.com/image.jpg)

Code

Inline

`inline code`
